# 关系数值系统 v0.1

本文件定义 IF 的核心关系数值。系统内部记录真实值，玩家看到的是感知反馈。数值服务于关系模拟，不服务于“看面板刷最优解”。

## 1. 真实值

真实值只在系统内部存在，不直接展示给玩家。

| 真实值 | 含义 | 主要来源 | 影响 |
| --- | --- | --- | --- |
| 真实好感 | 对对方的总体正向情绪 | 互动、陪伴、吸引、支持 | 升温、主动联系、复合可能 |
| 真实信任 | 相信对方说法和边界的程度 | 坦白、履约、查证结果、撒谎暴露 | 查证、解释接受度、危机强度 |
| 真实安全感 | 在关系中是否稳定、不被抛下 | 回复、公开关系、承诺、边界 | 冷战、吃醋、分手念头 |
| 真实压力 | 当前生活和关系负担 | 工作、上学、求职、家庭、经济、争吵 | 回复速度、爆发、撒谎、冷处理 |
| 真实矛盾 | 当前尚未解决的冲突强度 | 争吵、价值观差异、边界冲突 | 磨合、冷淡、分手危机 |
| 真实失望 | 对对方重复行为的累积失落 | 失约、忽视、敷衍、旧账 | 冷淡、分手念头、降低修复效果 |
| 生理吸引 | 对对方身体和亲密接近的吸引 | 外貌偏好、暧昧、亲密反馈 | 暧昧升温、亲密节奏 |
| 心理喜欢 | 对人格、陪伴和长期价值的喜欢 | 理解、支持、共同计划、价值观 | 长期稳定、修复关系 |
| 分手念头 | 结束关系的内在倾向 | 失望、原则错误、压力过载、替代对象 | 分手危机、冷淡、主动提分手 |
| 愧疚感 | 对自己隐瞒、伤害或失约的内疚 | 撒谎、失约、第三者、冷处理 | 坦白、补偿、反向防御 |
| 破绽值 | 某个隐瞒或谎言可被发现的线索强度 | 时间线矛盾、朋友圈、共同朋友、消费记录 | 查证成功、信任危机 |
| 关系满意度 | 对当前关系整体状态的主观评价 | 好感、信任、安全感、矛盾、压力 | 主动行为、忠诚、长期阶段 |

## 2. 感知值

玩家看到的不是数字，而是基于行为、表达、上下文和玩家自身性格生成的模糊反馈。

感知反馈示例：

- 你感觉对方还挺在意你。
- 对方最近似乎有点疏远。
- 你不确定她是真的忙还是不想回。
- 他这次解释让你感觉不太对。
- 你觉得这件事可能没那么简单，但还没有证据。
- 她表面上说没事，但语气比平时冷。
- 他愿意解释，不过你感觉有些细节被略过了。

感知值可能与真实值不同：

- 对方真实还喜欢，但因为加班和家庭压力显得冷淡。
- 对方真实已经动摇，但仍保持礼貌回复。
- 对方只是普通饭局，但玩家高不安导致误判。
- 对方确实隐瞒重点，但玩家因为信任较高暂时接受。

## 3. 真实值与感知值偏差

偏差来源：

- 信息不完整。
- 对方撒谎或隐瞒。
- 玩家过去经历造成高警觉。
- 共同朋友提供片面信息。
- 压力导致表达变差。
- 朋友圈或消费记录制造误导。

偏差后果：

- 玩家可能追问、忍住、冷淡、打电话、查证或试探。
- NPC 可能感到被不信任，也可能因为被追问暴露破绽。
- 误判如果反复发生，会变成关系中的长期记忆。

## 4. 关系阶段阈值规则

阶段由多维综合决定，不写死最终数值。

判断原则：

- 好感高但信任低，可能进入强吸引但高风险的暧昧或分分合合。
- 好感中等但信任、安全感、责任感稳定，可能进入长期稳定。
- 压力过载会暂时压低表达和回复，不必直接等于关系变差。
- 失望和未解决冲突持续累积，会降低修复效果。
- 生理吸引可以推动短期升温，但不能单独支撑长期关系。
- 心理喜欢和责任感影响稳定、复合和未来规划。

关键事件可以直接触发危机：

- 重大撒谎暴露。
- 隐瞒第三者接触。
- 原则性边界被突破。
- 公共场合严重羞辱。
- 多次拉黑或断联。
- 亲密边界被忽视。

原则性错误可以绕过慢速累积，直接进入分手危机。系统应记录原因，并让后续复合难度受到影响。

## 5. v0.1 计算口径

v0.1 只需要支持以下核心数值：

- 好感。
- 信任。
- 安全感。
- 压力。
- 矛盾。
- 失望。
- 生理吸引。
- 心理喜欢。
- 分手念头。
- 破绽值。
- 未解决冲突。

v0.1 不需要暴露公式给玩家。文档、事件和测试可以使用“上升、下降、明显上升、明显下降、进入账本”等描述，而不是具体数值加减。

## 6. 行为标签到数值的转换

事件不直接写“好感 +5”。事件先记录行为标签，再根据角色和上下文推导影响。

行为标签示例：

- 坦白。
- 追问。
- 回避。
- 撒谎。
- 冷处理。
- 查证。
- 道歉。
- 反击。
- 试探。
- 设边界。
- 逃避。
- 补偿。

同一个标签在不同角色身上后果不同：

- 对高边界敏感角色，“查证”可能明显降低信任。
- 对高不安角色，“坦白”可能提升安全感。
- 对高冲突回避角色，“追问”可能引发冷处理。
- 对高责任感角色，“补偿”更容易形成修复。
