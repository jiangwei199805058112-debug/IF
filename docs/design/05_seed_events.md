# v0.1 种子事件

本文件定义第一批 3 个种子事件。它们用于验证 v0.1 的核心机制，而不是构成完整事件库。

## 事件 1：对方很久没回消息

### 基础信息

- 事件ID：`MSG_001`
- 适用阶段：正在聊天 / 暧昧中 / 刚恋爱
- 触发条件：玩家发出消息后，NPC 在预期时间内未回复
- 参与角色：玩家、NPC，可选共同朋友
- v0.1 状态：启用

### 真实层

- 真实发生了什么：NPC 长时间没有回复玩家消息。
- 涉及角色：NPC，可能涉及同事、同学、家人或朋友。
- 真实对象：根据分支决定，可能是工作/课程/面试/家庭事务/异性朋友。
- 真实目的：
  - 真忙：NPC 确实处于工作、上课、面试、家庭事务或睡眠中。
  - 情绪回避：NPC 看到了消息，但暂时不想处理关系压力。
  - 隐瞒分支：NPC 正在参加不想让玩家知道的社交活动。
  - 试探分支：NPC 故意慢回，观察玩家反应。
- 是否撒谎：
  - 真忙分支：无或简单解释。
  - 情绪回避分支：可能模糊回答。
  - 隐瞒分支：可能半真半假或隐瞒重点。
  - 试探分支：通常不承认故意慢回。
- 是否有破绽：低到中。若朋友圈、共同朋友或时间线矛盾出现，破绽上升。
- 是否存在第三者：默认无，可由隐瞒分支低概率引出潜在第三者。
- 是否进入记忆账本：单次通常不进入；连续发生、解释矛盾或引发争吵时进入。
- 真实压力来源：工作、学校、求职、家庭、关系压力或社交压力。
- 真实风险：安全感下降、猜忌、冷淡、追问升级。

### 感知层

- 玩家看到什么：自己发出的消息长时间没有回复。微信没有已读机制，玩家不知道对方是否已经看到。
- 玩家不知道什么：NPC 是真忙、逃避、隐瞒、试探，还是手机不在身边。
- 玩家可能误判什么：
  - 把真忙误判成不在意。
  - 把逃避误判成背叛。
  - 把隐瞒误判成普通忙碌。
  - 把朋友圈动态误判成故意不回。
- 可用选项：
  - 追问：直接问“怎么这么久没回？”
  - 忍住：暂时不问，等待对方解释。
  - 冷淡：自己也降低回复热度。
  - 打电话：直接确认对方状态。
  - 试探：发一条轻描淡写但带情绪的消息。
  - 查证：查看朋友圈或共同朋友动态。
- 情绪反馈：不安、失落、猜忌、克制、赌气。
- 模糊提示文本：
  - “你不知道她有没有看到，只能盯着最后一条消息反复想。”
  - “他没有回复，但朋友圈里似乎有人刚发了聚餐照片。”
  - “你感觉这可能只是忙，但心里还是有点不舒服。”
- 信息可靠度：低。
- 是否可查证：可部分查证。

### 后果层

- 真实数值变化：
  - 真忙分支中，NPC 真实好感不一定下降，但压力可能上升。
  - 逃避分支中，真实矛盾和未解决冲突可能上升。
  - 隐瞒分支中，信任风险和破绽值上升。
  - 试探分支中，考验倾向被记录。
- 感知数值变化：
  - 玩家可能感到安全感下降。
  - 若 NPC 后续解释充分，感知信任可能恢复。
  - 若解释含糊，玩家感知破绽上升。
- 破绽值变化：
  - 无外部线索：无变化或轻微上升。
  - 朋友圈动态与解释冲突：明显上升。
  - 共同朋友提供不同说法：明显上升。
- 压力变化：
  - 玩家追问会提高 NPC 的关系压力。
  - 玩家忍住会降低即时冲突，但可能提高玩家内在不安。
  - 玩家冷淡会制造双向冷淡风险。
- 记忆记录：
  - “第一次长时间未回复”可作为轻量记忆。
  - 若同类事件重复，升级为“回复问题长期未解决”。
- 后续事件钩子：
  - `SOC_001` 异性饭局/神秘电话。
  - 查证朋友圈。
  - 第二天解释。
  - 冷淡延续。
- 可能触发的阶段变化：继续暧昧、升温后修复、冷淡、分手危机前兆。
- 修复机会：主动解释、承认忙乱、补偿沟通、明确回复习惯。

## 事件 2：异性饭局/神秘电话

### 基础信息

- 事件ID：`SOC_001`
- 适用阶段：暧昧中 / 刚恋爱 / 恋爱一段时间
- 触发条件：NPC 出现异性饭局、深夜电话、含糊社交安排或时间线缺口
- 参与角色：玩家、NPC、异性对象，可选共同朋友
- v0.1 状态：启用

### 真实层

- 真实发生了什么：NPC 与一名异性对象有饭局、电话或私下联系。
- 涉及角色：NPC、异性对象，可能有其他朋友或同事在场。
- 真实对象：
  - 普通同事/同学。
  - 关系较近的异性朋友。
  - 前任。
  - 潜在暧昧对象。
  - 工作或学习相关对象。
- 真实目的：
  - 普通社交。
  - 工作/学习需要。
  - 情绪倾诉。
  - 暧昧试探。
  - 隐瞒接触。
  - 骑驴找马。
- 是否撒谎：
  - 完全坦白：说明对象、原因、时间和边界。
  - 半真半假：说了饭局，但省略对象或真实目的。
  - 隐瞒重点：只说“和朋友吃饭”。
  - 完全撒谎：说在家、加班或睡了。
  - 反向指责：被问及时强调玩家不信任。
- 是否有破绽：中到高。取决于朋友圈照片、电话时间、消费记录、共同朋友和前后说法。
- 是否存在第三者：
  - 普通分支：无。
  - 暧昧分支：潜在。
  - 脚踏两只船分支：明确。
- 是否进入记忆账本：半真半假、隐瞒重点、完全撒谎、双标冲突时进入。
- 真实压力来源：社交压力、关系压力、工作压力、被发现风险。
- 真实风险：信任危机、第三者线索、边界重谈、分手危机。

### 感知层

- 玩家看到什么：
  - NPC 接到神秘电话后变得含糊。
  - NPC 说“普通朋友吃个饭”但没有细说。
  - 朋友圈或共同朋友提到一场饭局。
  - NPC 回复时间与解释不完全匹配。
- 玩家不知道什么：
  - 对方是否异性。
  - 是否单独见面。
  - 是否曾经暧昧或交往过。
  - 是否有其他人在场。
  - NPC 是否主动隐瞒。
- 玩家可能误判什么：
  - 普通工作饭局被误判成暧昧。
  - 暧昧试探被误判成普通朋友。
  - 对方的含糊来自怕吵架，而不一定来自背叛。
  - 双标角色可能忽略自己也有类似社交。
- 可用选项：
  - 接受：表示理解，但观察后续。
  - 追问：询问对象、时间和原因。
  - 吃醋：直接表达不舒服。
  - 查证：看朋友圈、问共同朋友或对时间线。
  - 冷战：不继续沟通，降低回复热度。
  - 设边界：提出异性饭局规则。
  - 反击：指出对方双标或隐瞒。
- 情绪反馈：不安、吃醋、愤怒、怀疑、委屈、控制冲动。
- 模糊提示文本：
  - “她说只是朋友，但没有说是谁。”
  - “他解释得很快，像是早就准备好了说法。”
  - “你没有证据，只是觉得这通电话让气氛变了。”
  - “你想起自己上次和异性朋友吃饭时，对方反应比现在更激烈。”
- 信息可靠度：低到中。
- 是否可查证：可部分查证，少数分支可直接查证。

### 后果层

- 真实数值变化：
  - 完全坦白且边界清楚：真实信任可能上升或保持。
  - 半真半假：短期冲突下降，破绽和长期风险上升。
  - 隐瞒重点：真实信任风险上升，愧疚感可能上升。
  - 完全撒谎：若未被发现，短期平稳；若暴露，直接进入高强度危机。
  - 反向指责：压力上升，未解决冲突上升。
- 感知数值变化：
  - 玩家可能感到安全感下降。
  - 若 NPC 主动说明边界，感知信任可恢复。
  - 若 NPC 回避对象身份，玩家感知破绽上升。
- 破绽值变化：
  - 前后说法不一致：明显上升。
  - 朋友圈照片出现异性对象：明显上升。
  - 共同朋友说法不同：明显上升。
  - 双重账号或两部手机线索：进入高风险钩子。
- 压力变化：
  - 玩家查证会提高双方压力。
  - NPC 隐瞒成功会短期降低冲突，但增加未来暴露压力。
  - 双标被指出会提高冲突强度。
- 记忆记录：
  - “异性饭局边界冲突”。
  - “对方解释含糊”。
  - “双标争议”。
  - “第三者线索”。
- 后续事件钩子：
  - 神秘电话再次出现。
  - 双重账号/两部手机线索。
  - 共同朋友补充信息。
  - 信任危机谈判。
  - 分手后发展第三者。
- 可能触发的阶段变化：冷淡、分手危机、分分合合倾向、边界重建后稳定。
- 修复机会：完整坦白、承认隐瞒、明确边界、减少单独接触、解释双标并修正。

## 事件 3：吵架后是否解决

### 基础信息

- 事件ID：`CONFLICT_001`
- 适用阶段：暧昧中 / 刚恋爱 / 恋爱一段时间
- 触发条件：消息、异性饭局、消费、公共场合或旧账引发争吵
- 参与角色：玩家、NPC，可选朋友或旁观者
- v0.1 状态：启用

### 真实层

- 真实发生了什么：双方发生冲突，必须决定当场解决、私下解决、冷处理或升级。
- 涉及角色：玩家、NPC，公共场合分支可涉及朋友或旁观者。
- 真实对象：冲突议题本身，例如回复、异性饭局、礼物、餐厅、隐私、旧账。
- 真实目的：
  - 解决问题。
  - 争取被理解。
  - 维护面子。
  - 逃避冲突。
  - 反击对方。
  - 逼对方表态。
- 是否撒谎：
  - 可能无撒谎，只是表达冲突。
  - 可能出现否认、模糊回答、隐瞒重点、反向指责。
- 是否有破绽：由前置事件决定。若争吵中说法改变，破绽上升。
- 是否存在第三者：通常无；若由 `SOC_001` 触发，可延续第三者线索。
- 是否进入记忆账本：高概率进入。若未解决、公共场合羞辱、拉黑或熬夜争吵，则必定进入。
- 真实压力来源：关系压力、睡眠不足、公共面子、工作/学校压力、家庭压力。
- 真实风险：冷战、拉黑、熬夜、第二天状态下降、分手危机。

### 感知层

- 玩家看到什么：
  - 对方语气变冷。
  - 对方说“算了，不想吵”。
  - 对方要求回家再说。
  - 对方在公共场合当场反驳。
  - 对方不回消息或拉黑。
- 玩家不知道什么：
  - 对方是真的想冷静，还是想逃避。
  - 对方是否已经产生分手念头。
  - 对方是否把这件事记入旧账。
  - 对方第二天是否因为熬夜状态下降。
- 玩家可能误判什么：
  - 把冷静误判成冷暴力。
  - 把逃避误判成成熟处理。
  - 把当场争论误判成不爱面子。
  - 把“没事了”误判成已经解决。
- 可用选项：
  - 当场说清：直接处理核心问题。
  - 回家说：避免公共场合升级。
  - 冷处理：暂时不回应。
  - 道歉：先承认自己的部分。
  - 反击：指出对方的问题。
  - 设边界：要求以后如何处理类似冲突。
  - 拉开距离：结束当天互动。
  - 补偿：用行动尝试修复。
- 情绪反馈：委屈、愤怒、疲惫、羞耻、不甘、心软。
- 模糊提示文本：
  - “她说没事，但这不像真的没事。”
  - “你们把问题压了下去，却没有真正说清。”
  - “已经很晚了，你感觉继续吵下去只会更糟。”
  - “朋友还在旁边，你能感觉到对方面子挂不住。”
- 信息可靠度：中。
- 是否可查证：主要通过后续行为查证。

### 后果层

- 真实数值变化：
  - 当场说清：可能降低未解决冲突，也可能因表达方式提高压力。
  - 回家说且真的解决：降低矛盾，提升信任。
  - 回家说但之后不谈：未解决冲突上升，旧账概率上升。
  - 冷处理：短期冲突下降，安全感和信任风险上升。
  - 拉黑：强烈提高分手危机和失望。
  - 熬夜争吵：第二天压力和判断力问题上升。
- 感知数值变化：
  - 玩家可能以为已经和好，但真实层仍有未解决冲突。
  - 玩家可能觉得对方不在乎，但对方真实只是过载。
  - 公共场合争吵会明显提升羞耻和失望感知。
- 破绽值变化：
  - 若争吵中说法改变，前置谎言破绽上升。
  - 若只是情绪冲突，破绽值可无变化。
- 压力变化：
  - 熬夜会让双方第二天状态下降。
  - 公共场合争吵会额外提高面子压力。
  - 私下解决如果成功，会降低长期压力。
- 记忆记录：
  - “公共场合争吵”。
  - “说好回家谈但没有谈”。
  - “冷战/拉黑”。
  - “熬夜争吵导致第二天状态差”。
  - “一次有效修复”。
- 后续事件钩子：
  - 第二天回复变慢。
  - 旧账翻出。
  - 主动道歉或补偿。
  - 分手危机谈判。
  - 朋友评价介入。
- 可能触发的阶段变化：磨合、冷淡、分手危机、修复后升温、分分合合倾向。
- 修复机会：明确问题、承认情绪伤害、约定下次处理方式、停止熬夜争吵、补偿但不替代沟通。
